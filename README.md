# 📘 Level 2: Math for Machine Learning

This repository contains the fundamental mathematical concepts required for Machine Learning. These topics form the foundation for understanding how ML algorithms learn from data, make predictions, and improve performance.

---

## 📚 Topics Covered

### 21. Linear Algebra

* Scalars, Vectors, and Matrices
* Matrix Operations
* Transpose
* Determinant
* Inverse Matrix
* Eigenvalues and Eigenvectors

**Applications:** Data representation, Neural Networks, PCA, Linear Regression.

---

### 22. Matrix Multiplication

* Matrix multiplication rules
* Dot product concept
* Real-world dataset operations

**Applications:** Neural Networks, Deep Learning, Image Processing.

---

### 23. Probability

* Basic probability concepts
* Probability calculations
* Event likelihood estimation

**Applications:** Classification, Spam Detection, Recommendation Systems.

---

### 24. Conditional Probability

* Probability given another event
* Event dependencies
* Probability-based predictions

**Applications:** Naive Bayes, Medical Diagnosis, Email Spam Detection.

---

### 25. Mean and Variance

* Mean (Average)
* Variance (Spread of Data)
* Data distribution analysis

**Applications:** Data Preprocessing, Feature Engineering, Outlier Detection.

---

### 26. Gradient Descent

* Optimization basics
* Learning rate
* Error minimization process

**Applications:** Training Machine Learning and Deep Learning models.

---

### 27. Loss Functions

* Mean Squared Error (MSE)
* Mean Absolute Error (MAE)
* Binary Cross Entropy

**Applications:** Measuring prediction error and model performance.

---

### 28. Derivatives

* Rate of change
* Slope calculation
* Gradient computation

**Applications:** Optimization and Backpropagation.

---

### 29. Optimization

* Parameter tuning
* Error minimization
* Model improvement techniques

**Popular Algorithms:**

* Gradient Descent
* Stochastic Gradient Descent (SGD)
* Adam Optimizer

---

### 30. Implement Gradient Descent

* Prediction
* Loss Calculation
* Gradient Calculation
* Weight Update
* Iterative Learning Process

**Outcome:** Model learns optimal parameters by minimizing error.

## 🎯 Learning Objectives

After completing this level, you will be able to:

* Understand the mathematical foundations of Machine Learning.
* Work with vectors and matrices.
* Apply probability concepts in ML problems.
* Understand loss functions and optimization techniques.
* Implement Gradient Descent from scratch.
* Prepare for ML internship interviews and beginner-level projects.
